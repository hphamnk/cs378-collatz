int collatz_find_cycle_length ( int  )

find the cycle length of a number recursively
if n = 1: return 1
if n is even: x = n / 2
- if x < 1 million and is in cache: get cycle length from cache
- else call this function and pass x in as parameter and +1 to the cycle length that it returns
if n is odd: x = n + (n >> 1) + 1
- if x < 1 million and is in cache: get cycle length from cache
- else call this function and pass x in as parameter and +2 to the cycle length that it returns
